# ?? Pizza Sales Analysis (SQL Project)

## ?? Project Overview
This project analyzes pizza sales data using SQL to derive insights such as revenue trends, popular pizzas, order distribution, and category performance.

## ?? Dataset
- orders.csv
- order_details.csv
- pizzas.csv
- pizza_types.csv

## ?? Tools Used
- MySQL
- MySQL Workbench
- GitHub

## ?? Analysis Performed
### Basic
- Total orders
- Total revenue
- Highest priced pizza
- Most ordered pizza size

### Intermediate
- Category-wise sales
- Hourly order distribution
- Average pizzas per day

### Advanced
- Revenue contribution by pizza type
- Cumulative revenue analysis
- Top pizzas by category (using window functions)

## ?? Key Skills Demonstrated
- SQL Joins
- Aggregations
- Subqueries
- Window Functions
- Business Insights
